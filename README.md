# PeaversRemembersYou

[![AddonSentry](https://api.addonsentry.io/api/public/repos/peavers-warcraft/PeaversRemembersYou/badge.svg)](https://addonsentry.io/dashboard/peavers-warcraft/PeaversRemembersYou)

A World of Warcraft addon that displays notifications when you group with players you've previously played with.

## Features

<!-- peavers:features -->
- Automatically tracks who you've grouped with
- Shows how long ago you last grouped with a player
- Indicates the previous group type (party, raid, etc.)
- Option to exclude guild members from notifications
- Configurable retention period for player history
- Low memory footprint
<!-- /peavers:features -->

## Usage

<!-- peavers:usage -->
The addon works automatically. When you join a group with someone you've grouped with before, you'll see a notification like: "You last grouped with Peavers in a raid 3 days ago!"

### Slash Commands

- `/pry` - Open the configuration panel
<!-- /peavers:usage -->

## Configuration

<!-- peavers:configuration -->
Access settings through `/pry` to customize:

- Retention period for player history
- Guild member exclusion
- Notification display settings
<!-- /peavers:configuration -->


## Installation

### Recommended: PeaversUpdater

Download and install [PeaversUpdater](https://github.com/peavers-warcraft/PeaversUpdater/releases/latest), the desktop updater for the whole Peavers collection. It installs PeaversRemembersYou together with its required dependencies and delivers updates before they reach CurseForge.

### Alternative: CurseForge

1. Download from [CurseForge](https://www.curseforge.com/wow/addons/peaversremembersyou)
2. Ensure [PeaversCommons](https://www.curseforge.com/wow/addons/peaverscommons) is also installed
3. Ensure [PeaversConfig](https://www.curseforge.com/wow/addons/peaversconfig) is also installed
4. Enable the addon on the character selection screen
